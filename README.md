# zork-ui

Play **Zork I** exactly as the original (authentic Z-machine engine), augmented with a
beautiful retro first-person 3D view of the room you're in — generated from the real game's
map data.

> Zork I source is MIT-licensed (Microsoft/Activision, 2025): https://github.com/historicalsource/zork1

Status: planning / scaffolding. See the implementation plan for the architecture and build
phases.
